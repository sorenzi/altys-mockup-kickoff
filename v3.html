Implementation · Built & running on simulator

Lesson kickoff — shipped

SwiftUI implementation of the V2 design. Three real screenshots from the actual built app on iPhone Air + iPad mini sims.

What's in the codebase now

  • Full rewrite of SellFlowLessonKickoffScreen.swift matching the V2 postcard design — 2-col grid only, vertical postcard cards, master-detail split on iPad landscape.
  • Topic colors use a new kickoffTint(for:) palette (sky/pink/golden/coral/lavender/peach/etc) so Ocean stays sky-blue across themes — the existing TopicCategory.chipColor was theme-dependent and collapsed several topics to mustard on the periwinkle theme.
  • iPad landscape detection uses the canonical GeometryReader { let isLandscapeIPad = isIPad && geo.size.width > geo.size.height } pattern from ABTestablePaywallView.
  • TopicCategory got slides(forAge:) (returns the 3-phrase bank) and description(forChildName:) (long blurb for the right panel).
  • ContentView.swift reverted back to AppEntryView — no test harness left behind.
  • Build clean on iPhone 17 Pro simulator.
